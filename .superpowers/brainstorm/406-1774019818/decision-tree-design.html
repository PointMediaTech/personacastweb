技術架構

SVG 層（前景 z-10）

  • 3 條 cubic Bézier 路徑
  • Origin 發光圓心 + 脈衝動畫
  • Framer Motion pathLength 生長動畫
  • Hover 交互：增亮 + 淡出其餘
  • 路徑標籤（SVG text）

Canvas 層（背景 z-0）

  • 粒子沿 3 條路徑曲線流動
  • 背景漂浮微粒（氛圍感）
  • Hover 時粒子加速聚集到目標路徑
  • requestAnimationFrame 迴圈

元件拆分

BranchingPaths (取代 NeuralCanvas + AgentDossiers)
├─ usePathParticles (Canvas hook: 粒子系統)
├─ DecisionTreeSVG (SVG: 路徑 + Origin + 動畫)
├─ OutcomeCard × 3 (磨砂玻璃終點卡片)
└─ HoverTooltip (條件顯示的細節浮層)

Hover 交互細節

Default：三條路徑同時可見，Optimal 路徑稍亮（opacity 0.8），Neutral 較暗（0.5）

Hover 某路徑：該路徑 opacity → 1.0 + strokeWidth 加粗 + glow filter。其餘路徑 → opacity 0.15。對應 OutcomeCard 放大 + 邊框增亮。

Tooltip：顯示在路徑中段附近，例如「此路徑觸發了 19 個衝突詞中的『造謠』機制」。300ms ease-out 淡入。
