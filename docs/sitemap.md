## 頁面說明

### 01 · 首頁 (Index)

主視覺入口頁，以沉浸式動態效果呈現平台願景：掌握變數、定義結局。包含 Hero 區域、Agent 卡片展示、數據視覺化與神經網路動畫等元素，引導訪客快速理解 PersonaCast 的核心價值。

### 02 · 應用場景 (Scenarios)

針對不同決策場景的深度展示：

- **政治選戰** — 72 小時輿論轉折點預演，模擬 T+32h 後的輿論走向
- **企業公關** — 危機降溫策略與議題置換路徑規劃
- **金融社會** — 情緒連鎖反應偵測與脈衝式傳播分析

### 03 · 技術深度 (Technology)

展示平台三大核心技術模組：

- **PersonaLab** — P_Final 性格公式引擎與 Zep 長期記憶系統
- **Casting Arena** — 雙面板並行模擬介面與時間軸推演技術
- **Knowledge Graph** — 80 節點、30 種關係類型的動態利益關係網

### 04 · 實戰案例 (Success Case)

以模擬回顧形式呈現平台實力：

- **案例 A** — 縣市長選舉中的議題置換成功路徑
- **案例 B** — 跨國企業公關危機的 56 小時黃金救援時間線

### 05 · 資源中心 (Resources)

提供開發者與決策者所需的深度資源：

- **OpenSpec** — 系統架構文件與 24 個 API 規格
- **產業白皮書** — AI 社會模擬的產業趨勢與未來展望
- **PersonaVault** — 企業專屬智庫建構指南

### 06 · 聯絡我們 / 預約演示 (Contact)

一對一專家演示預約入口，針對客戶專案提供初步模擬建議與方案評估。
